The goal is to learn a mapping from inputs $$x$$ to output $$y$$, given a labeled set of input-output pairs

$$
D = \{(x_i, y_i)\}^{N}_{i=1}
$$

$$D$$ is the __[[machine-learning.training-set]]__ and $$N$$ is the number of training examples.
