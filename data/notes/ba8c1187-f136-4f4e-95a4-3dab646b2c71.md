
[Dendron](https://dendron.so) is a note-taking tool.

Build site for local dev.

```
npx dendron-cli buildSiteV2 --wsRoot . --stage dev --serve
```

Build site for production

```
npx dendron-cli buildSiteV2 --wsRoot .  --stage prod
```
