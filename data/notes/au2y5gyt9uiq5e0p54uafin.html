Chess Tournament Pigeonhole Principle

Book: A Walk through Combinatorics

Theorem

A chess tournament has nnn participants, and any two players play one game against each other. Then it is true that in any given point of time, there are two players who have finished the same number of games.

Proof

There are nnn players. If there is a player AAA who has finished all of his n−1n - 1n−1 games, then all other players must have played at least one game (against player AAA).

So, the set of number of games played by players cannot have the values 000 and n−1n - 1n−1 in it simultaneously.

So, the number of games played has only n−1n - 1n−1 possible values. The number of players is nnn. By the Pigeonhole Principle, there must be two players who have played the same number of games.
