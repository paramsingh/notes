Number of Subsets of [n] is 2^n

Let [n][n][n] = {1,2,3,…,n}\{1, 2, 3, \dots, n\}{1,2,3,…,n}.

Theorem

For all positive integers n, the number of all subsets of [n][n][n] is 2n2^n2n.

Proof:

By induction

Base case: for nnn = 1, the statement is true as [1][1][1] has two subsets, the empty set ϕ\phiϕ and {1}\{1\}{1}.

Induction step:

Assume we know that the statement is true for nnn, and want to prove it for n+1n + 1n+1.

We divide all subsets of [n+1][n + 1][n+1] into two groups:

  1. Subsets containing n+1n + 1n+1
  2. Subsets not containing n+1n + 1n+1

Now, each subset not containing n+1n + 1n+1 is a subset of [n][n][n]. We know there are 2n2^n2n such sets, by the induction hypothesis.

Now, those subsets that contain n+1n + 1n+1 have n+1n + 1n+1 and a subset of [n][n][n]. The subset of [n][n][n] can be any subset, so the total number of subsets containing n+1n + 1n+1 is also 2n2^n2n.

Therefore, total number of subsets of [n+1]=2n+2n=2n+1[n + 1] = 2^n + 2^n = 2^{n+1}[n+1]=2n+2n=2n+1.

Hence proved.
